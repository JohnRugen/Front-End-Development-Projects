Audio Visualization & Playlist System - C#

Early this year I created a system within Unity that takes in data from audio files. This data was then manipulated to change various variables to create a system that could alter games in a variety of ways. Data could be manipulated to allow for game objects to scale / change depending on the music being played. Objects listened to a certain “beat” within a song and could then change when that beat occurs. For example, certain objects could listen for piano sounds whilst others listen for drums. This allows users to create games and programs that react to the music. This piece of work was released on the Unity Asset store. Source code & documentation is available on request.
